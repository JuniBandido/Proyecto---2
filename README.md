# Proyecto---2
Proyecto 2 del 2do Semestre - Programación Avanzada

Este Proyecto toma como base una tienda de materiales electricos en Morales, Izabal.
Donde nuestro principal objetivo fué ayduar a la compañia a tener un mejor control y manejo del sistema
Control sobre las ventas, los usuarios y productos existentes en bodega, todo mendiante un sistema creado en python y con una interfaz grafica usando la libreria de Tkiner
